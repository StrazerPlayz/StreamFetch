📥 YouTube Downloader (yt-dlp GUI)

A simple and powerful YouTube downloader built using yt-dlp, featuring an easy-to-use interface for downloading videos, audio, playlists, and more.

🚀 Features
🎬 Download videos in multiple qualities (including best available)
🎵 Extract audio (MP3, WAV, etc.)
📋 Queue multiple URLs
📊 Real-time download progress
📁 Automatic file organization
🎨 Clean and user-friendly interface
🔐 Member-Only Content Support

This downloader supports member-only videos and other restricted content — as long as you have access.

To download these:

You must be a member of the channel
Authentication (cookies/login) is required
